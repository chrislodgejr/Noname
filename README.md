# No Name Hospitality Group — White Interactive Preview

This is a static, Vercel-ready website preview. The GitHub version uses a CSS monogram mark so no binary logo file is required.

## Deploy to Vercel

1. Go to https://vercel.com/new
2. Import this GitHub repository: `chrislodgejr/Noname`
3. Framework preset: **Other**
4. Build command: leave blank
5. Output directory: leave blank
6. Deploy.

After deployment, Vercel will generate a test URL similar to:

`https://noname.vercel.app`

## Add restaurant website links

Open `script.js` and replace each `website: "#"` value with the correct restaurant URL.

Example:

```js
{
  name: "Frame Old City",
  website: "https://example.com"
}
```
